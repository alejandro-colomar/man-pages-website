The linux-man mailing list

The mailing list for the man-pages project is linux-man@vger.kernel.org. All patch submissions and project-related discussions should CC that list so that others can comment on patches and there is an archived record of the discussions relating to patches and project decisions.

Note that like other VGER mailing lists, HTML mail is rejected by the list server.

Subscription and archive

To subscribe to the list, send a message containing the following body to majordomo@vger.kernel.org: 

    subscribe linux-man

Searchable archives of this list can be found on gmane and MARC.
